# Bengaluru Restaurant Market Intelligence

## Objective
Analyse 844 Bengaluru restaurants across 8 locations to uncover
pricing, rating, cuisine, and location-level patterns.

## Key Findings
- Price does NOT drive ratings (r = -0.01)
- Popularity does not equal quality
- HSR Layout is the single best untapped market opportunity

## Approach
Data Cleaning → EDA → Statistical Testing → 15 Charts → Executive Summary

## Statistical Methods Used
Pearson, Spearman, Mann-Whitney U, Kruskal-Wallis, OLS Regression

## Tech Stack
Python, Pandas, Matplotlib, Seaborn, Plotly, SciPy, Statsmodels
